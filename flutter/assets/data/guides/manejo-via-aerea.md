# Manejo de Vía Aérea

## Evaluación Inicial (Mnemotecnia LEMON)

- **L** - Look externally (obesidad, barba, trauma facial)
- **E** - Evaluate 3-3-2 (apertura bucal >3 dedos, hioides >3 dedos, tiroides >2 dedos)
- **M** - Mallampati score (I-IV)
- **O** - Obstruction (cuerpo extraño, edema, absceso)
- **N** - Neck mobility (inmovilización cervical, artritis)

## Dispositivos para Vía Aérea

### Básicos (sin visualización directa)

- **Cánula orofaríngea (Guedel)**: Paciente inconsciente sin reflejo nauseoso
- **Cánula nasofaríngea**: Paciente consciente o con reflejo nauseoso
- **Bolsa-válvula-mascarilla (BVM)**: Ventilación con O2 100%, 10-15 L/min

### Avanzados

- **Mascarilla laríngea (LMA)**: Alternativa a ETI, fácil inserción
- **Combitubo**: Doble luz para esófago/tráquea
- **Tubo endotraqueal (TET)**: Vía aérea definitiva

## Intubación Endotraqueal

### Preparación

1. Equipo: laringoscopio, TET (7.0-8.0 mujer, 7.5-8.5 hombre), fiador, jeringa 10ml
2. Monitor: ECG, SpO2, ETCO2, PA
3. Medicación: sedante + bloqueante neuromuscular (secuencia rápida)

### Técnica Secuencia Rápida (RSI)

1. **Preoxigenación**: O2 100% x 3-5 min o 8 respiraciones profundas
2. **Inducción**: Etomidato 0.3 mg/kg o Ketamina 1-2 mg/kg IV
3. **Parálisis**: Succinilcolina 1-1.5 mg/kg o Rocuronio 1 mg/kg IV
4. **Presión cricoidea** (Sellick): solo si hay riesgo de aspiración
5. **Intubación**: Visualizar cuerdas vocales, avanzar TET 1-2 cm por debajo
6. **Inflar balón**: 5-10 ml de aire
7. **Confirmación**: Capnografía colorimétrica + auscultación + ETCO2 continuo
8. **Fijar y ventilar**: 6-8 ml/kg, 10-12 rpm

### Comprobación de Posición Correcta

| Método | Señal |
|--------|-------|
| Capnografía colorimétrica | Cambio de púrpura a amarillo |
| Capnografía cuantitativa | ETCO2 35-45 mmHg mantenido |
| Auscultación | Murmullos vesiculares simétricos + ausencia en epigastrio |
| Rx tórax | Punta TET 2-4 cm sobre carina |
| Condensación tubo | Sí, pero no confiable |

### Tasas de Dificultad

- **Mallampati I-II**: Intubación fácil
- **Mallampati III**: Intubación difícil
- **Mallampati IV**: Intubación muy difícil (preparar alternativa)

## Vía Aérea Quirúrgica

### Cricotirotomía (Emergencia)

**Indicación**: No se puede intubar, no se puede ventilar (CICO)

1. Identificar membrana cricotiroidea (entre cartílago tiroides y cricoides)
2. Incisión vertical piel, horizontal membrana
3. Insertar TET 5.0-6.0 o cánula de traqueostomía
4. Ventilar y confirmar

### Traqueostomía

**Indicación**: Vía aérea prolongada (>7-14 días), obstrucción supraglótica

## Complicaciones

- **Inmediatas**: Lesión dental, laceración mucosa, intubación esofágica, broncoaspiración, hipoxemia
- **Tardías**: Estenosis subglótica, fístula traqueoesofágica, neumonía asociada a VMI

## Extubación

- **Criterios**: Glasgow >8, reflejo nauseoso, ventilación espontánea, PaO2/FiO2 >150
- **Prueba de fuga**: Desinflar balón, oír fuga alrededor del tubo
- **Oxígeno post-extubación**: Venturi o cánula nasal

## Algoritmo de Vía Aérea Difícil (DAS)

```
Preoxigenar → Laringoscopia directa (≤2 intentos) →
  Falla → Dispositivo supraglótico →
    Falla → Ventilar con BVM (2 personas) →
      Falla → Cricotirotomía de emergencia
```
