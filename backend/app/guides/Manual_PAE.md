# Manual del Proceso de Atención de Enfermería (PAE)

El PAE es un método sistemático y organizado para administrar cuidados de enfermería individualizados.

## Fases del PAE

1. **Valoración:** Recolección de datos subjetivos y objetivos del paciente.
2. **Diagnóstico:** Identificación de problemas de salud basados en la taxonomía NANDA.
3. **Planificación:** Establecimiento de objetivos y prioridades utilizando los resultados NOC.
4. **Ejecución:** Puesta en práctica de las intervenciones planificadas utilizando la clasificación NIC.
5. **Evaluación:** Comparación del estado del paciente con los objetivos definidos (NOC) para medir el progreso.
