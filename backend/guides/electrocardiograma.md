# Interpretación de ECG

## Conceptos Básicos

### Papel de ECG

- **Velocidad**: 25 mm/segundo
- **Estandarización**: 10 mm/mV
- **Cuadro pequeño**: 0.04 seg (1 mm)
- **Cuadro grande**: 0.20 seg (5 mm)

### Derivaciones

| Grupo | Derivaciones | Visualizan |
|-------|-------------|------------|
| Extremidades (bipolares) | DI, DII, DIII | Plano frontal |
| Extremidades (aumentadas) | aVR, aVL, aVF | Plano frontal |
| Precordiales | V1-V6 | Plano horizontal |

## Sistema de Ráfaga para Interpretar

### 1. Ritmo

- ¿Hay onda P? → Ritmo sinusal vs. no sinusal
- ¿Ritmo regular o irregular?
- Frecuencia: 300 ÷ (cuadros grandes entre QRS)

### 2. Frecuencia (Adulto)

| Clasificación | Rango |
|---------------|-------|
| Normal | 60-100 lpm |
| Bradicardia | <60 lpm |
| Taquicardia | >100 lpm |

### 3. Eje Cardíaco

Normal: -30° a +90°

- **Desviación izquierda**: -30° a -90°
- **Desviación derecha**: +90° a +180°

Grosor QRS DI y aVF para determinar eje rápido.

### 4. Intervalos Normales

| Intervalo | Duración | Significado |
|-----------|----------|-------------|
| PR | 0.12-0.20 s | Conducción auriculoventricular |
| QRS | <0.12 s | Conducción intraventricular |
| QT | <440 ms (H), <460 ms (M) | Repolarización ventricular |
| QTc | (ver fórmula abajo) | QT corregido por FC |

**Fórmula de Bazett**: QTc = QT / √(RR en segundos)

### 5. Morfología

- **Onda P**: <2.5 mm altura, <3 mm duración
- **Complejo QRS**: <0.12 s, morfología progresión normal V1-V6
- **Segmento ST**: Plano a nivel isoeléctrico
- **Onda T**: Asimétrica, misma dirección que QRS
- **Onda U**: Puede ser normal (hipokalemia si prominente)

## Arritmias Comunes

### Taquiarritmias Supraventriculares

| Tipo | Características ECG |
|------|---------------------|
| **Taquicardia Sinusal** | P normal, FC >100, regular |
| **FA (Fibrilación Auricular)** | No P, ondas f líneas base irregular, QRS irregular |
| **Flutter Auricular** | Ondas F en diente de sierra, 300/min |
| **TSVP (reentrada nodal)** | Falta P retrógrada, QRS estrecho, regular, 150-250 |

### Taquiarritmias Ventriculares

| Tipo | Características ECG |
|------|---------------------|
| **TV (Taquicardia Ventricular)** | QRS ancho >0.12 s, regular, AV disociado |
| **FV (Fibrilación Ventricular)** | Actividad caótica, sin QRS |
| **Torsades de Pointes** | Taquicardia polimórfica con QT largo |

### Bradiarritmias

| Tipo | Características |
|------|-----------------|
| **Bradicardia sinusal** | P normal, FC <60 |
| **Bloqueo AV 1er grado** | PR >0.20 s fijo |
| **Bloqueo AV 2° grado Mobitz I** | PR prolonga progresivo hasta QRS caído |
| **Bloqueo AV 2° grado Mobitz II** | PR constante, QRS caído intermitente |
| **Bloqueo AV 3er grado** | Completo, P sin relación con QRS, ritmo de escape |

## Isquemia / IAM

### Patrón ECG por Localización

| Localización | Derivaciones afectadas | Arteria |
|-------------|----------------------|---------|
| Anterior extenso | V1-V6, DI, aVL | DA (descendente anterior) |
| Anterior septal | V1-V2 | DA septal |
| Anterior apical | V3-V4 | DA distal |
| Lateral | DI, aVL, V5-V6 | Circunfleja |
| Inferior | DII, DIII, aVF | CD (coronaria derecha) |
| Posterior | V7-V9 (ST deprimido V1-V2) | CD o Circunfleja |
| VD (ventrículo derecho) | V4R | CD proximal |

### Criterios de Sgarbossa (BRD + IAM)

- **ST elevación ≥1mm** en misma dirección → +5 puntos
- **ST depresión ≥1mm** en V1-V3 → +3 puntos
- **ST elevación ≥5mm** con QRS negativo → +2 puntos
- Total ≥3 = sospecha IAM

## Signos de Alarma en ECG

| Hallazgo | Significado |
|----------|-------------|
| ST elevado >1mm (2 derivaciones contiguas) | IAMCEST |
| ST deprimido + T invertida | IAMSEST/Isquemia subendocárdica |
| Q patológica (>0.04 s, >25% QRS) | Necrosis previa |
| Ondas T picadas "en carpa" | Hiperkalemia |
| QT largo >500 ms | Riesgo de Torsades |
| Epsilon (V1-V3) | Displasia VD arritmogénica |
| Ondas en Aplanada (V1) | Hipertrofia AD |
| P mitrale (bifásica DI, V1) | Hipertrofia AI |
