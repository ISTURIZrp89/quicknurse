# ACV Isquémico: Detección y Manejo Precoz en Urgencias

## Fuentes y Referencias
- **AHA/ASA 2019/2023** — Guidelines for Early Management of Acute Ischemic Stroke
- **ESO 2021** — European Stroke Organisation Guidelines
- **Código Ictus** — Protocolos nacionales (España, México, Colombia, Argentina)
- **UpToDate 2024** — "Acute ischemic stroke: Emergency evaluation and treatment"
- **NIHSS** — National Institutes of Health Stroke Scale (validada)

---

## Definición y Ventana Terapéutica
- **ACV Isquémico**: Déficit neurológico focal súbito por oclusión arterial cerebral (trombosis/embolía).
- **VENTANA TROMBOLISIS IV (alteplasa)**: **4.5 horas** desde último estado basal conocido (o despertar con síntomas).
- **VENTANA TROMBECTOMÍA MECÁNICA**: **6-24 horas** (selección por imagen: ASPECTS/perfusion mismatch).
- **TIEMPO ES CEREBRO**: 1.9 millones neuronas/minuto perdidos en ACV no tratado.

## Reconocimiento Prehospitalario / Triage (FAST / BE-FAST)

| Componente | Evaluación | Hallazgo anormal |
|------------|------------|------------------|
| **F**ace (Cara) | Sonreír / mostrar dientes | Asimetría facial / desviación comisura |
| **A**rm (Brazos) | Elevar ambos brazos 10 seg | Deriva / caída brazo (pronación) |
| **S**peech (Habla) | Repetir frase simple | Disartria / afasia / mutismo |
| **T**ime (Tiempo) | **¿Cuándo apareció?** | **Último estado basal conocido** |
| **B**alance (Equilibrio) | Marcha / Romberg | Caída, ataxia, vértigo |
| **E**yes (Visión) | Campos visuales / diplopía | Hemianopsia, visión doble, ceguera |

> **FAST positivo = CÓDIGO ICTUS**. Traslado preferente a **Unidad de Ictus (Stroke Unit)**.

## NIHSS — Escala estandarizada gravedad

| Ítem | Puntuación | Qué evalúa |
|------|------------|------------|
| 1a | 0-2 | Nivel conciencia |
| 1b | 0-2 | Preguntas (edad, mes) |
| 1c | 0-2 | Órdenes (cerrar ojos, puño) |
| 2 | 0-2 | Mirada conjugada |
| 3 | 0-3 | Campos visuales |
| 4 | 0-3 | Paresia facial |
| 5a/5b | 0-4 c/u | Fuerza brazo/ pierna (0-4) |
| 6a/6b | 0-4 c/u | Fuerza pierna/ brazo |
| 7 | 0-3 | Ataxia miembros |
| 8 | 0-2 | Sensibilidad |
| 9 | 0-3 | Lenguaje (afasia) |
| 10 | 0-2 | Disartria |
| 11 | 0-2 | Extinción/neglect |
| **TOTAL** | **0-42** | **≥25 = ACV grave / ≥16 = trombectomía candidata** |

> **NIHSS ≥6** → alta probabilidad oclusión vaso grande (LVO). **NIHSS ≥16** → candidatos prioritarios trombectomía.

## Flujo en Urgencias — **OBJETIVO Door-to-Needle ≤45 min**

| Tiempo | Acción | Responsable | Meta |
|--------|--------|-------------|------|
| **0 min** | Triage: FAST/BE-FAST + NIHSS | Enfermería triage | ≤10 min |
| **≤10 min** | **TC Craneal SIN CONTRASTE** (excluir hemorragia) | Radiólogo / Técnico | **≤20 min** |
| **≤20 min** | Lectura TC + decisión trombólisis | Neurólogo / Urgencias | ≤10 min |
| **≤30 min** | Labs: Coag (INR, aPTT, plaquetas), Glucemia, Hb, Creat, HCG si fértil | Laboratorio | ≤15 min |
| **≤45 min** | **ALTEPLASA IV** si candidato | Enfermería / Médico | **Door-to-Needle ≤45 min** |
| **≤60 min** | TC Angio / Perfusion (si LVO candidato) | Neurorradiología | ≤90 min |
| **≤90 min** | **Trombectomía mecánica** si LVO confirmado | Neurointervencionista | **Door-to-Groin ≤90 min** |

## Criterios Inclusión Alteplasa IV (AHA/ASA)

| INCLUSIÓN (Todas) | EXCLUSIÓN (Cualquiera) |
|-------------------|------------------------|
| ✅ Edad ≥18 años | ❌ Hemorragia en TC (o sospecha) |
| ✅ ACV isquémico clínico (NIHSS ≥4) | ❌ INR >1.7 / aPTT >40s / Plaquetas <100k |
| ✅ Último estado basal ≤4.5h | ❌ Glucemia <50 o >400 mg/dL |
| ✅ TC sin hemorragia / signos tempranos isquemia | ❌ Cirugía mayor / trauma craneal <3 meses |
| ✅ Consentimiento informado (verbal/escrito) | ❌ ACV previo 3 meses / IAM <3 meses |
| | ❌ Sangrado activo interno / úlcera activa |
| | ❌ Aneurisma intracraneal conocido / malformación AVM |
| | ❌ Embarazo (juicio individual) |
| | ❌ Anticoagulante oral directo (DOAC) <48h sin nivel |

> **EXTENSIÓN 4.5-9h**: Solo en ensayos clínicos (WAKE-UP, EXTEND, THAWS) con selección por imagen (mismatch perfusión-DWI).

## Protocolo Alteplasa IV (Activasa® / Actilyse®)

| Parámetro | Detalle |
|-----------|---------|
| **Dosis total** | **0.9 mg/kg** (máx 90 mg) |
| **Bolo inicial** | **10% (0.09 mg/kg)** en 1 min IV |
| **Infusión** | **90% restante en 60 min** |
| **Peso** | Usar peso real (si >100kg, usar 100kg para cálculo máx 90mg) |
| **Monitor** | FC, PA, NIHSS c/15min (1ª h), luego c/30min (6h), luego c/h (24h) |
| **PA objetivo** | **<180/105 mmHg** (antes y post-bolo). Si >180/105: Labetalol 10-20mg IV lento / Nicardipina 5mg/h IV |
| **Antídoto** | **Ácido tranexámico** NO indicado. Si hemorragia sintomática: **Crioprecipitados + Plaquetas + Factor VIIa** (juicio experto) |

---

## Manejo Post-Trombólisis (primeras 24h)

| Parámetro | Monitorización | Acción si anormal |
|-----------|----------------|-------------------|
| **NIHSS** | c/15min x1h → c/30min x6h → c/h x24h | Empeoramiento ≥4 pts → TC urgente (hemorragia) |
| **PA** | c/15min x2h → c/30min x6h → c/h | >180/105: Labetalol/Nicardipina. <140/90: reducir vasopresores |
| **Glucemia** | Ingreso + c/6h x24h | <70: bolo D50. >180: Insulina IV Slater |
| **Temp** | c/4h | >38°C: Paracetamol 1g IV + buscar foco |
| **Saturación** | Continua | <94%: O2 suplementario |
| **Deglución** | **NPO hasta test deglución** (agua 50mL) | Aspiración → SNG / Nutrición enteral temprana |

## Complicaciones — Red Flags

| Complicación | Frecuencia | Clave enfermería |
|--------------|------------|------------------|
| **Hemorragia intracraneal sintomática** | 2-6% (SITS-MOST) | NIHSS ↑≥4, cefalea, vómitos, descenso conciencia → TC URGENTE |
| **Edema cerebral maligno** | 1-5% (ACV gran territorio) | Día 1-5: hipertensión endocraneal → Descompresión descompresiva |
| **Angioedema orofaríngeo** | 1-5% (ACEi concomitante) | Edema lengua/labios/vía aérea → Epinefrina IM, corticoides, vía aérea |
| **Convulsiones** | 5-10% | Midazolam 2-5mg IV / Levetiracetam 1g IV |
| **Neumonía aspirativa** | 10-20% (disfagia) | Test deglución ANTES de alimentar, cabecera 30° |

---

## Enfermería en Stroke Unit (primeros 7 días)

| Día | Prioridades |
|-----|-------------|
| **0-1** | Monitor neuro estricto, control PA/glucemia/temp, movilización precoz (si estable), prevención TVP (HBPM 12h post-alteplasa), disfagia screening |
| **2-3** | Rehab multidisciplinar (fisioterapia, logopeda, OT), control factors riesgo (HTA, DM, FA, tabaco), educación paciente/familia |
| **4-7** | Plan alta / derivación rehab, ajuste tratamiento secundario (antiagregantes, estatinas, anticoagulación si FA), cita neurologo 7-14 días |

---

## Tratamiento secundario (prevención recurrencia)

| Etiología | Tratamiento | Inicio |
|-----------|-------------|--------|
| **Arterioesclerótica** (gran vaso) | **AAS 100-325 mg/día** + **Estatina alta intensidad** (Atorva 80mg/Rosuv 20-40mg) | 24h post-TC (si trombólisis) / Inmediato (si no) |
| **Cardioembólica** (FA, valvulopatía) | **Anticoagulación oral** (DOAC preferido: Apixaban 5mg c/12h / Rivaroxaban 20mg/día) | **NO <48h** si ACV leve; 7-14 días si moderado-grave (riesgo HT) |
| **Pequeño vaso** (lacunar) | AAS + Estatina + Control HTA estricto (<130/80) | Inmediato |
| **Criptogénica** | AAS + Estatina + Monitor Holter 24-72h / Ecocardiograma transesofágico | Inmediato + estudio etiológico |
