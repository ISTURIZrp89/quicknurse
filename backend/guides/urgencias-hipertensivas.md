# Neumonía/Gripe/Covid: Diagnóstico y Criterios de Ingreso

## Fuentes y Referencias
- **Guía ESC/ERS Neumonía adquirida comunitaria 2023**
- **OMS/OMS Influenza — Manejo estacional**
- **OMS COVID-19 living guidance 2024**
- **Criterios IDSA/ATS CURB-65 y PSI/PORT**

## 1. Clasificación
| Teoría | Gravedad | Perfil |
|--------|----------|--------|
| Leve | NAC ambulatoria | Afebril/leve, no disnea, comorbilidad baja |
| Moderada | NAC hospitalización | CURB-65 = 1, no shock |
| Grave | NAC en UCI | CURB-65 = 2+, shock, fracaso orgánico |

## 2. Criterios Hospitalización
- CURB-65 ≥ 2
- Infección multilobular bilateral
|-------------------|
| Derrame pleural | derrame, empiema |
| Neumotórax | radiológico + |
| Sepsis | lactato > 2 |
| Sepsis grave | shock |
| TEP | D-dímero + angioTC |
| Pericarditis | dolor + ECG |
| Boerhaave | radiocontraste/endoscopia |

## 3. Criterios UCI (CURB-65 ≥ 3 o PSI IV-V)
- Requerimiento de apoyo respiratorio (FiO2 > 50 %, VM no invasiva)
- Hemodinámica inestable o shock
- Insuficiencia multiorgánica
- Neutropenia grave (< 0,5 x10^3)
