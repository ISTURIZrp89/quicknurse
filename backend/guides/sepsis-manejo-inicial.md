# Sepsis: Detección y Manejo Inicial en Urgencias

## Fuentes y Referencias
- **Surviving Sepsis Campaign (SSC) 2021/2022** — Guías internacionales actualizadas
- **SEMI/SEMICYUC** — Sociedad Española de Medicina Interna / Cuidados Intensivos
- **SSC 1-Hour Bundle** — Protocolo tiempo-dependiente
- **qSOFA / SOFA** — Scores de severidad validados
- **UpToDate 2024** — "Sepsis and septic shock in adults: Diagnosis and management"

---

## Definiciones clave
- **SEPSIS**: Disfunción orgánica potencialmente mortal causada por respuesta desregulada del huésped a una infección (SOFA ≥2).
- **SHOCK SÉPTICO**: Sepsis con hipotensión persistente que requiere vasopresores para mantener PAM ≥65 mmHg y lactato >2 mmol/L tras reposición volémica adecuada.
- **SEPSIS SIN SHOCK**: Sepsis sin criterios de shock.

---

## Cribado temprano (qSOFA) — <2 min en triage

| Criterio | Punto |
|----------|-------|
| Alteración nivel conciencia (Glasgow <15) | 1 |
| PA Sistólica ≤100 mmHg | 1 |
| FR ≥22 rpm | 1 |

**qSOFA ≥2** → **Alto riesgo sepsis** → Activar código sepsis + bundle 1h.

> ⚠️ qSOFA tiene **alta especificidad** (≈90%) pero **baja sensibilidad** (≈60%). **NO sustituye juicio clínico**. Usar SOFA completo si hay dudas.

---

## Código Sepsis — Bundle 1 hora (SSC 2021)

| Paso | Acción | Tiempo máx | Detalle |
|------|--------|------------|---------|
| **1** | **Hemocultivos** (2 juegos: aerobio + anaerobio) | **0 min** | Antes de ATB. Incluir cultivo de orina, esputo, catéter si sospecha. |
| **2** | **Lactato** (arterial/venoso) | **0 min** | Si >2 mmol/L → repetir a 2-4h. Meta: clearance >20% o normalización. |
| **3** | **Antibióticos empíricos IV** | **≤60 min** | Amplio espectro según foco sospechoso. Ver tabla abajo. |
| **4** | **Reposición volémica** (Cristaloides 30 mL/kg) | **≤60 min** | Si hipotensión/lactato >4. Reevaluar cada 500 mL (PAM, FC, diuresis, ecografía). |
| **5** | **Vasopresores** si PAM <65 tras fluidos | **≤60 min** | Noradrenalina 1ª línea. Objetivo PAM ≥65 mmHg. |
| **6** | **Reevaluación hemodinámica** | Continuo | FC, PAM, diuresis, capilar, ecografía POCUS, ScvO₂ si disponible. |

## Antibioterapia empírica (inicio ≤1 hora)

| Foco sospechoso | Regimen recomendado | Alternativas / Alergia |
|-----------------|---------------------|------------------------|
| **Desconocido / Nosocomial grave** | Meropenem 1g c/8h IV **+** Vancomicina 15-20 mg/kg c/8-12h IV (ajustar trough) | Piperacilina-Tazobactam 4.5g c/6h + Vancomicina |
| **Fuente abdominal** | Meropenem 1g c/8h **+** Metronidazol 500mg c/8h (si no carbapenem) | Pip-Tazo + Metronidazol; Ciprofloxacino + Metronidazol |
| **Neumonía adquirida comunidad grave** | Ceftriaxona 2g c/24h **+** Azitromicina 500mg c/24h (o Levofloxacino 750mg) | Moxifloxacino; Meropenem si MRSA |
| **Neumonía nosocomial / VAP** | Meropenem 2g c/8h **+** Vancomicina/Linezolid **+** Amikacina (espectro Pseudomonas) | Ceftazidima/Avibactam; Cefepime |
| **Infección urinaria complicada / Pielonefritis** | Ceftriaxona 2g c/24h **+** Gentamicina 5-7mg/kg (dosis única) | Pip-Tazo; Meropenem si ESBL |
| **Meningitis bacteriana** | Ceftriaxona 2g c/12h **+** Vancomicina 15-20mg/kg c/8-12h **+** Dexametasona 0.15mg/kg c/6h (antes/ATB) | Meropenem si alergia grave |

> **Ajuste según cultivo y sensibilidades a 48-72h (de-escalation).**

## Reposición volémica — estrategia

| Solución | Dosis | Comentario |
|----------|-------|------------|
| **Suero Salino 0.9%** / Ringer Lactato | 30 mL/kg bolus inicial | Ringer Lactato preferido (menor acidosis hiperclorémica) |
| **Albumina 5%** | Si gran reposición (>30 mL/kg) o hipoalbuminemia | No superior a cristaloides en SSC; usar si refractario |
| **Monitorización** | Cada 500 mL: PAM, FC, SatO₂, diuresis, ecografía IVC/IVC colapsabilidad | Meta: PAM ≥65, diuresis >0.5 mL/kg/h, lactato ↓ |

---

## Vasopresores — escalera SSC

| 1ª línea | **Noradrenalina** 0.05-0.5 mcg/kg/min (títralo a PAM ≥65) |
|---|---|
| 2ª línea (añadir) | **Vasopresina** 0.03 U/min (fijo, no titular) — reduce dosis noradrenalina |
| 3ª línea (refractario) | **Adrenalina** 0.05-0.5 mcg/kg/min (efecto β1 inotrópico + α) |
| 4ª línea | **Dopamina** (solo si bradicardia asociada / bajo riesgo arritmia) — **NO 1ª línea** |

> ⚠️ **Noradrenalina SIEMPRE por CVC** (riesgo necrosis extravasación). Si no hay CVC: poner CVC urgente o usar vasopresina periférica corta duración mientras se canaliza.

---

## Corticoides en shock séptico refractario
- **Indicación**: Shock que **requiere vasopresores** tras reposición adecuada + ATB.
- **Dosis**: **Hidrocortisona 50 mg IV c/6h** (o 200 mg/día infusión continua) x 5-7 días, desescalar.
- **NO rutinario** si responde a fluidos/vasopresores.

---

## Soporte de órganos diana

| Órgano | Intervención enfermería |
|--------|------------------------|
| **Respiratorio** | Protección vía aérea, ventilación protectora (Vt 6 mL/kg PBW, Pplat <30), PEEP según FiO₂, pronación si PaO₂/FiO₂ <150 |
| **Renal** | Monitor diuresis hora, balance hídrico, evitar nefrotóxicos, considerar CRRT si AOKD estadio 3 |
| **Hematológico** | Hemoglobina objetivo 7-9 g/dL (trasfusión restrictiva), plaquetas >10k si sangrado / >50k si procedimiento |
| **Metabólico** | Glucemia 140-180 mg/dL (insulina IV si >180), nutrición enteral temprana ≤48h |

---

## Red flags — **ESCALAR INMEDIATO**
- Lactato ≥4 mmol/L o no clearance >20% a 2h
- PAM <65 tras 30 mL/kg + noradrenalina >0.5 mcg/kg/min
- Nueva falla orgánica (oliguria <0.5 mL/kg/h x 2h, confusión aguda, PaO₂/FiO₂ <150)
- Coagulopatía (INR >1.5, plaquetas <100k, fibrinógeno <1.5 g/L)
- Acidosis metabólica (pH <7.2, BE < -5)

| ✅ | Acción | Responsable | Registro |
|----|--------|-------------|----------|
| | Código sepsis activado | Triage | Hora activación |
| | 2 hemocultivos + otros cultivos | Enfermería | Hora extracción |
| | Lactato inicial | Lab / Enfermería | Valor + hora |
| | ATB IV primera dosis | Enfermería / Médico | Fármaco, dosis, hora |
| | 30 mL/kg cristaloides | Enfermería | Volumen, tiempo, PAM c/500mL |

| | Balance hídrico estricto | Enfermería | Ingresos/egresos cada hora |
