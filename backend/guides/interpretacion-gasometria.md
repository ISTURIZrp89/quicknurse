## Valores de Referencia (Adulto, nivel mar, 37°C)

| Parámetro | Rango normal | Unidades |
|-----------|--------------|----------|
| **pH** | **7.35 - 7.45** | — |
| **PaCO₂** | **35 - 45** | mmHg |
| **PaO₂** | **80 - 100** | mmHg |
| **HCO₃⁻ (calculado)** | **22 - 26** | mEq/L (mmol/L) |
| **BE (Exceso de base)** | **-2 a +2** | mEq/L |
| **SatO₂** | **>95%** | % |
| **Na⁺** | 135-145 | mEq/L |
| **K⁺** | 3.5-5.0 | mEq/L |
| **Cl⁻** | 98-106 | mEq/L |
| **Glucosa** | 70-110 | mg/dL |
| **Lactato** | **<2** (ideal <1) | mmol/L |
| **FiO₂** | 0.21 (aire ambiente) | — |

> **Nota**: PaO₂ esperado = (FiO₂ × 713) - (PaCO₂ / 0.8) [ecuación gas alveolar]. Gradiente A-a normal ≤15-20 mmHg (joven) / ≤25 (anciano).

## Método Sistemático — 6 PASOS (ROME / Método fisiológico)

### **PASO 1: Oxigenación**
| Parámetro | Interpretación |
|-----------|----------------|
| **PaO₂ <60 mmHg** | **Hipoxemia severa** (insuficiencia respiratoria tipo I) |
| **PaO₂ 60-80** | Hipoxemia moderada |
| **SatO₂ <90%** | Hipoxemia clínica (curva disociación Hb) |
| **PaO₂/FiO₂ (Índice P/F)** | **>300**: Normal / **200-300**: ALI / **100-200**: SDRA moderado / **<100**: SDRA severo |

### **PASO 2: pH — Acidosis vs Alcalosis**
- **pH <7.35** → **ACIDOSIS**
- **pH >7.45** → **ALCALOSIS**
- **pH 7.35-7.45** → Normal (pero puede haber trastorno mixto compensado)

### **PASO 3: Componente Primario (Respiratorio vs Metabólico)**

| Trastorno primario | pH | PaCO₂ | HCO₃⁻ | Causa típica |
|--------------------|----|-------|-------|--------------|
| **Acidosis respiratoria** | ↓ | **↑** | Normal/↑ (comp) | Hipoventilación (EPOC, sobredosis opioides, NMI, obesidad) |
| **Alcalosis respiratoria** | ↑ | **↓** | Normal/↓ (comp) | Hiperventilación (ansiedad, sepsis temprana, embolia, dolor, IAM) |
| **Acidosis metabólica** | ↓ | Normal/↓ (comp) | **↓** | Láctica, cetoacidosis, renal, tóxicos (MUDPILES), diarrea |
| **Alcalosis metabólica** | ↑ | Normal/↑ (comp) | **↑** | Vómitos, diuréticos asa, exceso alcalino, hipopotasemia, hiperaldosteronismo |

### **PASO 4: Compensación — ¿Es adecuada? (Fórmulas de Winter)**

| Trastorno PRIMARY | Compensación ESPERADA | Fórmula |
|-------------------|----------------------|---------|
| **Acidosis metabólica** | Hiperventilación (↓PaCO₂) | **PaCO₂ esperada = (1.5 × HCO₃⁻) + 8 ± 2** |
| **Alcalosis metabólica** | Hipoventilación (↑PaCO₂) | **PaCO₂ esperada = (0.7 × HCO₃⁻) + 20 ± 5** (máx ~55 mmHg) |
| **Acidosis respiratoria aguda** | Renal (↑HCO₃⁻) | **ΔHCO₃⁻ = 0.1 × ΔPaCO₂** (aguda) / **0.35 × ΔPaCO₂** (crónica) |
| **Alcalosis respiratoria aguda** | Renal (↓HCO₃⁻) | **ΔHCO₃⁻ = -0.2 × ΔPaCO₂** (aguda) / **-0.4 × ΔPaCO₂** (crónica) |

> **Si PaCO₂ real ≠ esperada ±5 mmHg → Trastorno MIXTO (respiratorio + metabólico simultáneo)**

### **PASO 5: Anion Gap (AG) — Solo en Acidosis Metabólica**
| Fórmula | Normal | Interpretación |
|---------|--------|----------------|
| **AG = Na⁺ - (Cl⁻ + HCO₃⁻)** | **8-12 mEq/L** (ajustar por albúmina: +2.5 por cada 1 g/dL <4) | **AG ↑ (>12)**: Acidosis metabólica con AG elevado (MUDPILES) |
| **Delta-Delta** | ΔAG / ΔHCO₃⁻ | **Ratio 1-2**: AG puro. **<1**: AG + acidosis normal AG concurrente. **>2**: AG + alcalosis metabólica concurrente |

**MUDPILES** (Acidosis metabólica AG elevado):
- **M**etanol / **U**remia / **D**KA / **P**araldehído (raro) / **I**nhalación (cianuro, CO) / **L**actato (shock, sepsis) / **E**tanol/glicol / **S**alicilatos

### **PASO 6: Contexto Clínico + Lactato + Electrolitos**
- **Lactato >4 mmol/L** → Hipoperfusión / Shock / Sepsis (clearance serial)
- **K⁺**: Corregir según pH (↑0.6 mEq/L por cada 0.1 ↓pH en acidosis metabólica)
- **Glucosa**: DKA >250 + AG↑ + cetonas; HHS >600 + pH normal

## Casos Clínicos Típicos Enfermería

### **Caso 1: EPOC exacerbado**
- **Gasometría**: pH 7.28 / PaCO₂ 65 / HCO₃⁻ 30 / PaO₂ 55 (FiO₂ 0.28)
- **Análisis**: pH↓ + PaCO₂↑ + HCO₃⁻↑ → **Acidosis respiratoria aguda sobre crónica** (compensación renal parcial)
- **Acción**: **NO O₂ alto** (riesgo CO₂ narco). Venturi 24-28% target Sat 88-92%. NIV (BiPAP) si pH<7.35. Broncodilatadores, corticoides, ATB si infección.

### **Caso 2: DKA (Cetoacidosis diabética)**
- **Gasometría**: pH 7.15 / PaCO₂ 28 / HCO₃⁻ 10 / PaO₂ 95 / AG 28 / Glucosa 480 / Lactato 3
- **Análisis**: pH↓ + HCO₃⁻↓↓ + PaCO₂↓ (comp) + AG↑ → **Acidosis metabólica AG elevado (DKA) + compensación respiratoria adecuada** (PaCO₂ esperada = 1.5×10+8 = 23±2; real 28 → leve insuficiencia respiratoria / fatiga)
- **Acción**: Fluidoterapia agresiva (SF 1L/h), Insulina IV 0.1 U/kg/h, K⁺ reposición (si <5.3), Bicarbonato SOLO si pH<6.9.

### **Caso 3: Sepsis severa / Shock séptico**
- **Gasometría**: pH 7.22 / PaCO₂ 30 / HCO₃⁻ 12 / PaO₂ 65 (FiO₂ 0.5) / Lactato 6.8
- **Análisis**: pH↓ + HCO₃⁻↓ + PaCO₂↓ + AG↑ (lactato) → **Acidosis metabólica láctica (AG↑) + alcalosis respiratoria compensatoria** (PaCO₂ esperada 1.5×12+8=26±2; real 30 ≈ adecuada)
- **Acción**: Código sepsis (ver guía sepsis). Fluidos 30 mL/kg, ATB <1h, Noradrenalina PAM≥65, Lactato serial (meta clearance >20%), NIV/IMV si falla respiratoria.

### **Caso 4: Sobredosis Opioides**
- **Gasometría**: pH 7.18 / PaCO₂ 70 / HCO₃⁻ 26 / PaO₂ 45 (aire)
- **Análisis**: pH↓ + PaCO₂↑↑ + HCO₃⁻ normal → **Acidosis respiratoria aguda pura** (sin compensación renal — aguda)
- **Acción**: **Naloxona 0.4mg IV** (repetir q2-3min hasta despertar/ventilar). Ventilación asistida (BVM / IMV). Monitor apnea recurrencia (vida media naloxona < opioides).

### **Caso 5: Vómitos prolongados / Deshidratación**
- **Gasometría**: pH 7.52 / PaCO₂ 48 / HCO₃⁻ 38 / PaO₂ 92 / Cl⁻ 88 / K⁺ 2.8
- **Análisis**: pH↑ + HCO₃⁻↑↑ + PaCO₂↑ (comp) → **Alcalosis metabólica hipoclorémica hipopotasémica + compensación respiratoria** (PaCO₂ esperada 0.7×38+20=46±5; real 48 ✓)
- **Acción**: SF + KCl reposición (20-40 mEq/L a 10-20 mEq/h), Ondansetrón, buscar causa (obstrucción, píloro, bulimia). **NO bicarbonato**.

## Gasometría Venosa (VBG) vs Arterial (ABG)

| Parámetro | Correlación VBG ↔ ABG | Uso clínico |
|-----------|----------------------|-------------|
| **pH** | VBG ≈ ABG - 0.02 a -0.05 | **Buena** para acidosis/alcalosis |
| **PaCO₂ / PvCO₂** | PvCO₂ ≈ PaCO₂ + 4-6 mmHg | **Aceptable** tendencia |
| **HCO₃⁻ / BE** | Muy similar | **Buena** |
| **PaO₂ / PvO₂** | **NO correlaciona** (PvO₂ ~40 mmHg) | **NO usar** para oxigenación |
| **Lactato** | Venoso ≈ Arterial (ligero ↑ venoso) | **Buena** para screening sepsis |

> **VBG útil** para seguimiento pH/CO₂/HCO₃⁻/Lactato en pacientes sin necesidad de punción arterial repetida. **NO para PaO₂/P/F**.

## Enfermería — Procedimiento Punción Arterial

| Paso | Detalle |
|------|---------|
| **1. Preparación** | Jeringa heparizada (0.1 mL heparina 1000 UI/mL) / Kit ABG pre-heparinizado. Eliminar burbujas. |
| **2. Allen Test** | Compresión radial+ulnar → apertura ulnar → retorno color <5-10 seg = **+** (seguro). Si - → usar femoral/braquial. |
| **3. Localización** | Arteria radial (1ª elección), braquial, femoral. Palpar pulso. |
| **4. Punción** | Ángulo 30-45°, avance hasta retorno sangre arterial (brillante, pulsátil). **NO aspirar** (deja llenar solo). |
| **5. Post-punción** | **Compresión FIRME 5 min** (10 min si anticoagulado). Verificar hemostasia. Venda compresiva. |
| **6. Muestra** | Expulsar burbujas, tapar, mezclar suave (rodar), **hielo si >10 min a análisis**. |
| **7. Documentación** | Hora, sitio, FiO₂, ventilación, circunstancias, enfermera. |
